# SUNDAYJUICE

Creative / play / listening / sketch / ritual repo.

This repo is for low-pressure creative use: dropping URLs, making crates, testing listening workflows, and keeping the Sunday/play side alive without turning it into a technical machine build.

## What this is

- A creative utility / ritual surface.
- A low-energy crate-grabbing helper.
- A place for listening/sketch workflows.
- A Sunday/play lane, not a hardware engineering lane.

## What this is not

- Not HARDWAREJUICE.
- Not the local AI audio-engine repo.
- Not a hardware controller namespace.
- Not a place for Max/MSP, OSC bridges, firmware, or model benchmarks.
- Not a monetisation project.
- Not an impressive app for its own sake.

Index.html

## Current role

Crate Grabber:

```text
paste URLs → clean/generate yt-dlp command → copy/paste into A-Shell or later Mac workflow
```

## Boundary with HARDWAREJUICE

SUNDAYJUICE = creative input/play/ritual.

HARDWAREJUICE = technical machine/build repo.

If a change involves MIDI, OSC, Max, RAVE, AFTER, MRT2, model benchmarks, hardware mapping, firmware, or local neural audio architecture, it belongs in HARDWAREJUICE.

## Cleanup rule

Optimise for usability only:

- clearer labels;
- fewer dead buttons;
- obvious file paths;
- one main entry point;
- no duplicate/conflicting purpose.

Do not add features until the current flow is clean.
